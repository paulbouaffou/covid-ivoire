# COVID IVOIRE

Information platform on the situation of CORONAVIRUS (COVID-19) in CÃ´te d'Ivoire

### Requirements

* [Python 3.x+](https://www.python.org/downloads/)
* [PIP (Python Dependency Manager)](https://pip.pypa.io/en/stable/installing/)

### Clone project

```$ git clone https://github.com/paulbouaffou/covid-ivoire```

### Installation
* `pip install -r requirements.txt`
* `flask run --reload # optionally you add --port=5000`

### Quickstart the tool

* ```$ FLASK_APP=app.py```
* ```$ export FLASK_ENV=development```
* ```$ flask run```

### On localhost
```$ localhost:5000```

### On Heroku server
```$ https://covid-ivoire.herokuapp.com```
